# WeatherApp

Приложение с прогнозом погоды, показывает прогноз на 24 часа и на ближайшие семь дней по координатам с устройства либо с указанием города.

## Технологии

```ruby
UIKit
```
```ruby
UICollectionView
```
```ruby
UITableView
```
```ruby
CoreLocation
```
